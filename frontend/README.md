# Timbre Frontend

Next.js 14 (App Router) UI.

## Dev

```bash
npm install
npm run dev   # port 3000
```

## Pages

| Route | Description |
|-------|-------------|
| `/login` | Sign in |
| `/signup` | Create account |
| `/canvas` | Main editor — generate voice-matched drafts |
| `/onboarding` | 4-stage voice fingerprint capture |
| `/studio` | View / edit voice profile |
| `/spaces` | Manage spaces |
